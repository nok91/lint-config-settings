## `@nokon/eslint-config`

### Usage
```sh
    npm install @nokon/eslint-config --dev
```

Create the corresponding files and extend the configurations Eslint (.eslintrc):

```
    {
        "extends": [
            "@nokon/eslint-config"
        ],
    }
```

```
    const eslintConfig = require('@nokon/eslint-config');
```
